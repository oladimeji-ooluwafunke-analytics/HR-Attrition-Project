## 📊 HR Attrition Analysis 

## Project Overview
This project focuses on analyzing employee attrition using Power BI. The goal is to identify key factors contributing to employee turnover and provide data-driven insights that can help organizations improve retention strategies.
## The dashboard presents a comprehensive view of workforce metrics, attrition patterns, and key drivers influencing employee exits.

## Objectives
    Analyze overall employee attrition rate
    Identify major factors driving employee turnover
    Explore attrition trends across different demographics and job roles
    Provide actionable insights to support HR decision-making

## Dataset

## Key Insights

* Employees working overtime show the highest attrition rate (53.59%), indicating work-life balance issues.
* Low-income employees have the highest turnover, suggesting compensation plays a major role.
* Certain job roles such as Laboratory Technician and Sales Executive experience higher attrition.
* Employees living farther from work show increased attrition, highlighting the impact of commute distance.
* Job satisfaction is directly linked to retention, with lower satisfaction leading to higher attrition.
  
## Recommendations

* Reduce excessive overtime to improve work-life balance
* Review and improve compensation, especially for low-income employees
* Focus on high-attrition roles with targeted retention strategies
* Introduce flexible work options to address commute challenges
* Improve employee engagement through career development and recognition programs
     
##  Tools Used
* Power BI – Data visualization and dashboard creation
* Power Query – Data transformation and cleaning
* DAX (Data Analysis Expressions) – Measures and calculations


## Conclusion
This project demonstrates how data visualization can be used to uncover critical workforce insights. By identifying the main drivers of attrition, organizations can take proactive steps to improve employee satisfaction and retention.
